# Medical Image Federated Learning with Explainable AI

This project demonstrates a federated learning framework for medical image classification with explainable AI techniques.

## Objectives
- Train models on distributed medical datasets
- Preserve patient privacy using Federated Learning
- Improve model transparency using Explainable AI (XAI)

## Technologies
- Python
- TensorFlow / PyTorch
- Federated Learning (FedAvg)
- SHAP / LIME explanations

## Dataset
Example dataset: ISIC 2019 skin lesion dataset.

## Features
- Medical image classification
- Federated learning across multiple clients
- Explainable AI visualization
